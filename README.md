# hexzm
SPA with Vue and Graphql

## How run project

step 1: download project
```
git clone https://github.com/mohamadrezamosalli/hexzm.git
```

step 2: install packages
```
npm install 
    OR
npm i
```

step 3: run project

just run server
```
npm run server
```

just run client
```
npm run client
```

run client and server in development mode
```
npm run dev
```
